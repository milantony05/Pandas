# Pandas Study Plan Solutions

This folder contains Python solution files for the LeetCode study plan [Introduction to Pandas](https://leetcode.com/studyplan/introduction-to-pandas).

Each `.py` file matches one exercise from the study plan:

- Create a DataFrame from List
- Get the Size of a DataFrame
- Display the First Three Rows
- Select Data
- Create a New Column
- Drop Duplicate Rows
- Drop Missing Data
- Modify Columns
- Rename Columns
- Change Data Type
- Fill Missing Data
- Reshape Data: Concatenate
- Reshape Data: Pivot
- Reshape Data: Melt
- Method Chaining

Use the file names to browse the solutions quickly or compare approaches across exercises.
